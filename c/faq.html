CBRN-TRAINER FAQ

Über das Projekt

Der CBRN-TRAINER ist ein Simulationsprojekt für die Ausbildung von Einsatzkräften im Bereich CBRN (Chemisch, Biologisch, Radiologisch, Nuklear). Das Tool ermöglicht es Ausbildern, verschiedene Messgeräte und Szenarien zu simulieren, ohne echte Gefahrstoffe oder Strahlenquellen zu benötigen.

Wie funktioniert es?

Für Ausbilder (Trainer)

  1. Erstellen Sie eine neue Session oder loggen Sie sich in eine bestehende ein
  2. Nutzen Sie die Schieberegler, um die Werte für verschiedene Parameter einzustellen
  3. Teilen Sie die Session-ID mit Ihren Teilnehmern oder nutzen Sie den QR-Code
  4. Verwenden Sie die vordefinierten Szenarien oder erstellen Sie eigene Situationen

Für Teilnehmer (Trainee)

  1. Erhalten Sie die Session-ID von Ihrem Ausbilder
  2. Geben Sie die ID ein oder scannen Sie den QR-Code
  3. Wählen Sie das zu verwendende Messgerät
  4. Beobachten Sie die Messwerte in Echtzeit

Verfügbare Messgeräte

  • CO Warngerät (PAC 6500)
  • Gas Multiwarngerät (X-AM 8000)
  • Dosisleistungsmessgerät (Automess)
  • Dosiswarngerät (Automess)
  • DL-Warner (Automess)

Vordefinierte Szenarien

  • Brandszenario: Simuliert typische Gaskonzentrationen bei einem Brand
  • Stadtgas (Raum): Simuliert eine leichte Gasansammlung in einem Raum
  • Stadtgas (Leck): Simuliert ein aktives Gasleck mit kritischen Werten
